Beautiful Css

For a lot of developers writing good maintainable CSS can be hard. This shouldnâ€™t be the case. Beautiful-Css is a gem to help clean up your CSS. It removes duplication, removes unused styles, and arranges you CSS so you know exactly where every style is located

Beautiful Css follows the idea of adding selectors to styles not styles to selectors. By writing css in this fashion any developer new to your style sheet will know exactly every ever property is located because they are in alphabetical order and are only ever listed one.
